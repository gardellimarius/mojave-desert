# mojave-desert
Mojave desert landing page

This is a simple landing page built with Pug, Scss, JavaScript and ScrollTrigger.js.

You'll need NodeJS in order run this in dev mode.
Navigate to this project folder, use npm install with no args, it'll just install all your dependencies locally.
Use npx webpack-dev-server to run it in dev mode, or npm run build to run it in production.
